Installing & Testing CodeBlocks

In CSE 103, you will be writing your code on your computer using an application called CodeBLocks. CodeBlocks is as an IDE, or Integrated Development Environment, which means that it allows us to edit our code, browse our files, run our programs and debug them, all in the same place. This handout is intended to take you through installing CodeBlocks on your own computer.

Installation

To get started, download and install the following version of CodeBlocks:
